# iot-cloud-health-monitoring-system
IoT and Cloud Computing based smart healthcare monitoring system which accepts health related inputs using sensors, analyses it uses Arduino microcontroller, stores information on and recovers information from the cloud and shows live and stored information on the mobile android app
- The Arduino reads the inputs coming in from the sensors.
- It send the data from the sensors to the bluetooth module.
- The bluetooth module is connected to the mobile phone and the Android app.
- The app interprets the information and shows it on the screen.
- The buttons on the app send the information being captured to the databse on the cloud.
- Additionally, the app can also create user accounts and fetch saved data from the cloud.
